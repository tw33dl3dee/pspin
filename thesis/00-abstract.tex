\begin{abstract}

  В работе рассмотрены основные проблемы, возникающие при генерации множества состояний
  для моделей большого размера, и существующие подходы к их решению. Предложен способ их
  решения за счет параллельной генерации состояний с хранением на нескольких узлах
  вычислительной сети.

  В рамках решения поставленной задачи разработан и реализован параллельный алгоритм
  генерации состояний с двумя способами распределения состояний между узлами. Реализован
  транслятор языка описания моделей Promela в используемое генератором состояний
  представление. Проведены эксперименты по сравнению разработанного ПО с существующим и по
  сравнению разработанных способов распределения состояний.

\end{abstract}
